# New Year Countdown Web Application written in VueJS. Just for fun

This is a New Year countdown project. It includes harmless 'Easter Eggs' (memes/Rickrolls) that trigger after a certain number of clicks. A consent modal is included to ensure users are aware of external redirects.

## How to install
1. Install Node.js
2. Clone this repository
3. get into `new-year` directory of the cloned project
4. run `npm install`

## How to run
inside `new-year` run `npm run serve` for dev build and `npm run build` run for production
